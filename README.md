# Mathematics-of-Finance
An introduction to Mathematics of Finance with Python.
1. We impement the Black-Schole model for European call options via Monte Carlo simulations and we compare the performance of the model to the exact solution.
2. It seems that we need aproximately 50 000 simulated paths to be close enough to the analytical solution.
3. Given the market price of an option, we compute the implied volatility from the Black-Scholes model.
